Прошивка ESP32‑C3 через браузер і USB (WebSerial)
Встановлення через HTTP заборонено!
Відкрийте цей сайт через HTTPS.
Ваш браузер не підтримується!
Спробуйте Chrome, Edge або Opera на комп’ютері.
Підключайте ESP32‑C3 кабелем USB та обирайте прошивку нижче.
Як прошити
  1. Підключіть плату ESP32‑C3 до цього комп’ютера (USB‑кабель має бути data, не лише зарядка).
  2. Натисніть кнопку Connect під потрібною прошивкою.
  3. Оберіть COM‑порт у вікні браузера.
  4. Якщо плата не визначається — введіть її в режим прошивки: BOOT затиснути → RESET натиснути/відпустити → BOOT відпустити.
